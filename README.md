# Простое WPF-приложение для расчета стоимости мобильной связи.

Основные функции

    Расчет стоимости по двум тарифам

    Автоматический учет бесплатных минут

    Генерация чека в текстовом файле

    Сохранение чеков на рабочий стол

Тарифы
Тариф 1

    200 бесплатных минут

    0.7 ₽/минута в пакете

    1.6 ₽/минута сверх пакета

Тариф 2

    100 бесплатных минут

    0.3 ₽/минута в пакете

    1.6 ₽/минута сверх пакета

Как использовать

    Введите количество минут

    Выберите тариф

    Нажмите "Рассчитать"

    Нажмите "Сформировать квитанцию" для чека
